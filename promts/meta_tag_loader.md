# Скрипт загрузки мета тегов

Реализуй консольную команду laravel которая будет загружать в базу данных мететеги из переданного csv файла

## Исходные данные

Скрипт по умолчанию работает с файлом public/loadet_data/new_meta_tags.csv но должна быть предусмотрена возможность передеть другой файл параметром.

Скрипт работает в 2 режимах: загрузка категорий, загрузка товаров по умолчанию загрузка категорий но должна быть возможность выбора режима

## Алгоритм работы скрипта

1. Скрипт читает данные из переданного csv файла и запускает цикл по строкам этого файла. При этом пропускает первую строку как заголовок.
2. Если скрипт работает в режиме "загрузка категорий" то ищем первую строку в которой не пустая ячейка под номером 2 (заголовок Страницы категорий и тегов)

- Далее берем значение ячейки 2 и определяем с какой моделью будем работать. Определяем по следующему алгоритму:
    - https://florida46.ru/catalog/cvety-v-korobkax - модель Category ( признак наличие /catalog/ )
    - https://florida46.ru/tags/gipsofila - модель ProductTag ( признак наличие /tags/ )

- Из значения ячейки 2 выделяем slug пример: https://florida46.ru/catalog/miagkie-igruski slug = miagkie-igruski
- Ищем элемент определенной ранее модели с таким slug и меняем в нем поля title, seo_title, seo_description. Значения для них берем из обробатываемой строки ячейки 4,5,6 соответствено.

3. Если скрипт работает в режиме "загрузка товаров" то ищем первую строку в которой не пустая ячейка под номером 3 (заголовок Страницы товаров)

Так же определяем slug по описанному правилу. Пример https://florida46.ru/tovar/51-odnogolovaia-roza-miks slug = 51-odnogolovaia-roza-miks

Ищем элемент модели Product с таким slug и меняем в нем поля title, seo_title, seo_description. Значения для них берем из обробатываемой строки ячейки 4,5,6 соответствено.

Скрипт должен выводить процесс выполнения и финальный отчет по проделанной работе.
